# 🧠 Algorithm Visualizer

An **interactive and visually engaging** platform to explore and understand the inner workings of popular algorithms — from sorting and searching to graph and recursion problems.

Built with **React**, **TailwindCSS**, and **Vite** for a fast and modern development experience.

---

## 🚀 Features

### 🔢 Sorting Algorithms
- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  

### 🔍 Searching Algorithms
- Linear Search  
- Binary Search  

### 🗺️ Pathfinding Algorithms
- Dijkstra’s Algorithm  
- Breadth-First Search (BFS)  
- Depth-First Search (DFS)  

### 🧩 Graph Algorithms
- Kruskal’s Minimum Spanning Tree (MST)  
- Prim’s Minimum Spanning Tree (MST)  
- Topological Sort  

### ♻️ Recursion Visualizations
- Tower of Hanoi  
- N-Queens Problem  

---

## ⚙️ Getting Started

Follow these simple steps to set up and run the project locally:

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/<your-username>/algorithm-visualizer.git
cd algorithm-visualizer
````

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

Now open your browser and navigate to 👉 **[http://localhost:5173](http://localhost:5173)**

---

## 🏗️ Build for Production

To create an optimized production build:

```bash
npm run build
```
