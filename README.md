# Algorithm Visualizer

A modern, interactive algorithm visualizer built with React and TailwindCSS.

## Features

- **Sorting Algorithms**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort
- **Searching Algorithms**: Linear Search, Binary Search
- **Pathfinding Algorithms**: Dijkstra's Algorithm, BFS, DFS
- **Graph Algorithms**: Kruskal's MST, Prim's MST, Topological Sort
- **Recursion**: Tower of Hanoi, N-Queens

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 18
- React Router v6
- TailwindCSS
- Lucide Icons
- Vite
